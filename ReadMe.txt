Program has been created to run at set intervals determined by the operator using Windows Scheduler.
A Batch file has been creaeted to run the program written in Python.
The program stores a list of possible exercises, and when started it gives a random exercise to the end user.
It then asks if they would like to carry out another execise, and if 'Yes' it will give anotehr random exercise.
If prompted again it will not give more than two exercises at one time.
